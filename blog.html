Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. For example:
1. document.getElementById('element-id').
2. document.getElementsByClassName('class-name').
3. document.getElementsByTagName('tag-name').
4. document.querySelector('selector').
5. document.querySelectorAll('selector').

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: When we use the innerHTML property, it reads both the HTML markup and the text content of the element.
2. innerText: When we use the innerText property, it reads only the text content of the element.
3. textContent: When we use the textContent property, it ignores all HTML tags and returns only the text. It reads text as it is in the markup. It also returns all text, whether it's rendered on screen or not. Also, textContent only deals with the raw text and doesn't account for styles.

Question-3: What is event delegation in the DOM?

In DOM, "event delegation" is a technique where we attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

In DOM, "event bubbling" is the process where when an event occurs on an element, it triggers not only on that element but also on its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

1. To create a new element, we can use the document.createElement() method.
2. To add the element to the DOM, we can use the appendChild() method.
3. To remove an element from the DOM, we can use the removeChild() method.
